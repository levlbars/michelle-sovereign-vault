Architecture

Two-graph split, never rejoined at inference:

Language Model (W8A16 · Hexagon NPU)
  • 32 layers, hidden_size=2560, vocab=248,320
  • Split into 5 ONNX chunks → 5 HTP context binaries
  • INT8 weights, FP16 activations
  • W8A16 encoding patch covers all 4 self-attention MatMul inputs per layer
  • Chunks gate-passed: FP32 bit-exact (constant-fold via onnxslim)
Vision Tower (FP16 · Adreno GPU)
  • FP16, not quantized — intentional (8-bit introduces noisy embeddings)
  • Frozen during SFT training
  • Communicates with LM via projected embeddings at <|vision_start|>..<|vision_end|> slots
  • Runs on Adreno GPU for continuous monitoring use cases

Intended Use

On-device multimodal reasoning on Snapdragon 8 Gen 3 devices. Target applications: continuous visual monitoring, on-device reasoning without cloud dependency, privacy-sensitive inference tasks.

Limitations

  • Optimized for SM8650 (Snapdragon 8 Gen 3) — HTP context binaries are device-specific.
  • Prefill benchmark is the measured value; decode throughput pending a decode-shaped graph compile.
  • Per-run context reload overhead present in current benchmarks — persistent-context runner will improve end-to-end latency.
  • Vision tower fine-tuning not included in current SFT (text-only data, tower frozen).

Access

Weights, distillation scripts, W8A16 quantization pipeline, tower-split compiler, and LANE configs are proprietary and not publicly distributed. The benchmarks above are verifiable on equivalent hardware with licensed artifacts.
